# Neon Requiem: Afterglitch

A cyberpunk arcade survival shooter built with **Phaser 3 + TypeScript + Vite**.

## Gameplay

You are a rogue runner trapped in a corrupted city grid.

- Move with **WASD** or **Arrow keys**
- Shoot with **Space**
- Survive endless enemy waves
- Score points per enemy destroyed
- Lose health on contact
- On death, press **R** to restart

## Features (MVP)

- Neon cyberpunk visual style and HUD
- Responsive movement + shooting loop
- Enemy spawn waves that scale with score
- Health + score tracking
- Game-over state with restart
- Polished UI shell around the canvas

## Tech Stack

- [Phaser 3](https://phaser.io/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vite.dev/)
